# To Do List

## Instalação

### Pré-requisitos
- **PHP**
- **Composer**
- **MySQL**
- **Node.js e npm**

### Passo a Passo

1. **Clone o repositório**
```bash
git@github.com:BelicioMoreira/todo-list.git
```
2. **Instale o Composer**
```bash
composer install
```
3. **Instale o npm**
```bash
npm install vite
```
4. **Copie o arquivo de configuração .env**
```bash
cp .env.example .env
```
5. **Gere a chave do Laravel**
```bash
php artisan key:generate
```
6. **Execute as migrações**
```bash
php artisan migrate
```
7. **Inicio o servidor**
```bash
php artisan serve
```
8. **Iinicie o npm**
```bash
npm run dev
```
9. **Acesse a URL no seu navegador**
```bash
http://localhost:8080
```
